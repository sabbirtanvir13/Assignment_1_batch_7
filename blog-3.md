How do Generics allow you to build reusable components and functions that stay strictly typed regardless of the data structures passed in? 

TypeScript এর অন্যতম শক্তিশালী ফিচার হলো Generics। এটি আমাদের এমন functions, classes  তৈরি করতে সাহায্য করে যেগুলো যেকোনো data type এর সাথে কাজ করতে পারে এবং type safety বজায় রাখে। ফলে কোড হয় reusable flexible 
Generics হলো এমন একটি system যেখানে আমরা type কে parameter হিসেবে pass করতে পারি এবং
Generics ছাড়া সাধারণভাবে function লিখলে type fixed হয়ে যায়
